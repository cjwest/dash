#Dash

This module provides an interface for administrators to monitor and act upon events.

## What is Dash?

A dashboard for monitoring and controlling REST based systems.

## Installation

### Install Guzzle.

To use Composer to install Guzzle, visit http://guzzle.readthedocs.org/en/latest/overview.html#installation
